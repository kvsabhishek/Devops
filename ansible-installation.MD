# Installing PIP
```
curl https://bootstrap.pypa.io/get-pip.py -o get-pip.py
```
```
python3 get-pip.py --user
```

# Installing Ansible
```
python3 -m pip install --user ansible
```
```
python3 -m pip install --user ansible-core
```

# Upgrading Ansible
```
python3 -m pip install --upgrade --user ansible
```
